# Data Analysis Projects

## Overview
This folder contains applied data analysis projects focused on extracting insights from real-world datasets using Python and exploratory data analysis techniques. Each project demonstrates the ability to clean, analyze, visualize, and interpret data to support data-driven decision making.

## Objectives
- Transform raw data into structured, analyzable formats
- Identify patterns, trends, and relationships within datasets
- Communicate insights through clear visualizations and summaries
- Apply analytical reasoning to real-world problems

## Projects Included
- **TMDB Movie Data Analysis**  
  Analysis of movie metadata to explore relationships between budget, revenue, popularity, and audience ratings.

- **Bikeshare Usage Analysis**  
  Exploration of transportation usage patterns to identify peak usage times, customer behavior trends, and operational insights.

- **Customer Segmentation Analysis**  
  Clustering and descriptive analysis to identify distinct customer groups and behavioral characteristics.

## Tools & Technologies
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## Key Skills Demonstrated
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Statistical summaries
- Insight-driven storytelling
